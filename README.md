# AgroMind Agent (العقل الزراعي) 🌿🤖

AgroMind Agent is an advanced autonomous customer support system specifically designed for the agricultural sector. It leverages state-of-the-art AI orchestration to provide technical support, farming advice, and customer service.

## 📁 Project Structure (هيكلة المشروع)

```text
agro-mind-agent/
│
├── backend/        # FastAPI + API logic (الخلفية البرمجية)
├── agent/          # AI Orchestrator (العقل المدبر للعميل)
├── tools/          # All defined functions/tools (الأدوات والوظائف)
├── rag/            # RAG system + vector DB (نظام استرجاع المعلومات)
├── memory/         # Customer memory system (ذاكرة العميل)
├── db/             # Database models + setup (قواعد البيانات)
├── frontend/       # Streamlit / Gradio UI (واجهة المستخدم)
├── safety/         # Escalation + safety logic (منطق الأمان والتصعيد)
├── data/           # Knowledge files (PDFs, docs) (ملفات المعرفة)
├── tests/          # Testing suite (الاختبارات)
├── main.py         # Application entry point (نقطة انطلاق التطبيق)
└── README.md       # Project documentation (توثيق المشروع)
```

## 🛠 Features

- **Autonomous Decision Making**: Powered by a central AI orchestrator.
- **RAG Integration**: Retrieves accurate agricultural data from local documents.
- **Personalized Memory**: Remembers previous interactions to provide better context.
- **Safety First**: Includes logic to handle escalations and ensure safe AI responses.
